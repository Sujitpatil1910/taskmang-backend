Setup Instructions
### 1. Clone the Repository
```bash
git clone https://github.com/Sujitpatil1910/taskmang-backend.git
cd task-manager
```

### 2. Set Up the Backend
```bash
cd backend
npm install
node server.js
```


### 3. Set Up the Frontend
```bash
cd client
npm install
npm start
```
